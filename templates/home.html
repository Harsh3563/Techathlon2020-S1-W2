NTU IEEE Workshop Series: Image Caption Generator using ML

This NTU IEEE Workshop Series is to create a full-functioning Image Captioning web application. The user is able to upload an image and our application generates a caption for the image within seconds! This workshop makes use of Machine Learning to generate the image captions, it uses Flask as a backend framework to make API calls, and uses React as a frontend framework to create the User Interface of this web application.

Some key points to note:
1- Our model depends on the data, so, it cannot predict the words that are out of its vocabulary. We used a small dataset consisting of 8000 images. For production-level models, we need to train on datasets larger than 100,000 images which can produce better accuracy models.
2- Try to use outdoor images like nature scene and animals with jpg and jpeg image format to get accurate result.
